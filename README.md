# Baby Name Combinator

A simple web-based tool to combine and compare different first, middle, and last names for babies.

## Features

- **Three Scrolling Dials**: Browse through first, middle, and last names with intuitive up/down controls
- **Live Name Preview**: See the full name combination as you dial through options
- **Customizable Name Lists**: Paste your own lists of names into the text areas
- **Real-time Updates**: Changes to name lists are immediately reflected in the dials

## How to Use

1. Enter your list of first names in the "First Names" text area (one per line)
2. Enter your list of middle names in the "Middle Names" text area (one per line)
3. Enter your list of last names in the "Last Names" text area (one per line)
4. Use the up/down arrows on each dial to scroll through the name options
5. The full name combination will update as you dial through selections

## Technical Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Getting Started

Simply open `index.html` in your web browser to start using the name combinator.
